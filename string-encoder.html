Encoding Information:

Base64: A binary-to-text encoding scheme that represents binary data in an ASCII string format. Commonly used for encoding data in emails and web applications.

URI Component: Percent-encoding used in URLs to encode special characters. Converts characters to %XX format where XX is the hexadecimal value.

Binary to Hex: Converts each character to its hexadecimal representation. Each byte is represented as two hexadecimal digits.

HTML Entities: Converts special characters to their HTML entity equivalents (e.g., < becomes &lt;). Essential for displaying text safely in HTML documents.

Unicode Escape: Converts characters to their Unicode escape sequences (e.g., \\u0041 for 'A'). Useful for JavaScript strings and JSON data.

Emoji Encode/Decode: Converts between emoji characters and their Unicode representations or shortcode names (e.g., 😀 ↔ :grinning_face:).

JSON Escape: Escapes special characters for safe inclusion in JSON strings. Handles quotes, backslashes, newlines, and control characters.
